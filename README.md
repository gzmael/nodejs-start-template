## Features

- NodeJS
- TypeScript
- Express
- Cors
- Jest (teste de integração e unitário)
- Eslint + Prettier
- Commitizen + Husky
- Build with Babel
- Prisma
- Cache with Redis
- Account Routes + Authenticate with JWT
